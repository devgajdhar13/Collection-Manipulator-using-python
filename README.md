# 🎓 Student Data Organizer

A simple **Python console application** to manage student records using
Python collection data types. This project demonstrates CRUD operations
through a menu-driven interface.

------------------------------------------------------------------------

## 📝 Description

The application allows users to add, display, update, and delete student
records while maintaining a unique list of subjects. It demonstrates the
practical use of **List, Dictionary, Set, Tuple, loops, conditional
statements, and exception handling**.

------------------------------------------------------------------------

## ✨ Features

-   ➕ Add Student
-   📋 Display All Students
-   ✏️ Update Student Information
-   🗑️ Delete Student
-   📚 Display Subjects Offered
-   🚪 Exit Program

------------------------------------------------------------------------

## 🛠️ Data Structures Used

  Data Structure   Purpose
  ---------------- ----------------------------------------------
  📋 List          Stores the order of Student IDs
  📖 Dictionary    Stores complete student information
  📚 Set           Stores unique subjects
  🆔 Tuple         Stores Student ID and Date of Birth together

## 🚀 How to Run

### Prerequisites

-   Python 3.x

### Steps

1.  Clone or download this repository.
2.  Open the project folder.
3.  Run the Python file:

``` bash
python "Collection Manipulator.py"
```

4.  Select an option from the menu.
5.  Follow the prompts to manage student records.
6.  Choose **6** to exit the program.
